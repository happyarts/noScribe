# Voxtral-Messungen (Nacht auf den 20.07.2026)

Referenz: Audiotest2/referenz/hart_780-900_REFERENZ.txt (handkorrigiert, 422 Woerter)
WER = Wortfehlerrate, CER = Zeichenfehlerrate auf leerzeichenfreiem Text.
WER hoch + CER niedrig = blosse Schreibvariante; beide hoch = echter Hoerfehler.
Overlap = wie viele Woerter der gleichzeitig sprechenden zweiten Stimme erkannt wurden.
